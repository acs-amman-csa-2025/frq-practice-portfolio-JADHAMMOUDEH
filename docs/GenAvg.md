## Assignment purpose

The purpose was trying to generate an array of random integers then calculating the average of the values in the array.
## Problem statement

Create a class called `GenAvg` that generates an array of 50 random integers, then calculates and prints their average. Make sure to create `GenAvg.md` in the docs folder and answer our reflection prompts there as well.
